You are a senior full-stack engineer.

Build a production-quality Progressive Web App (PWA) called VIDYASETU.

The application should be architected like a real SaaS product with clean folder structure, reusable components, mock backend services, role-based flows, responsive design, and presentation-ready UI.

DO NOT build a simple prototype.

Build the application as if it will later evolve into a production educational platform.

---------------------------------------
PROJECT OVERVIEW
---------------------------------------

Vidyasetu is a WhatsApp-first AI-assisted assessment infrastructure for low-resource classrooms.

Problem:

Teachers struggle to identify learning gaps because checking worksheets manually is time-consuming.

Solution:

Teachers create assignments.

Students or parents upload solved worksheet images.

AI extracts answers, detects misconceptions, groups students, and generates remediation recommendations.

Teachers receive classroom insights and intervention suggestions.
